Processo de instalação do Gestor IDSS (e de outras ferramentas) para consultas futuras.


Gestor IDSS

Criar um diretório c:\app
Criar um subdiretório oracle (c:\app\oracle) e descompactar o conteúdo do arquivo instantclient-basiclite-nt-19.6.0.0.0dbru.zip dentro dele
Copiar os arquivos tnsnames.ora e sqlnet.ora para \oracle\instant
Copiar a pasta GestorIDSS para c:\app
Criar um subdiretório cloud-connet (C:\App\cloud-connect) e copiar a pasta G:\Drives compartilhados\BlendusSEG\wallet_dblend3 para ele
Criar duas variáveis de Ambiente, em Variáveis do Sistema: ORACLE_HOME e TNS_ADMIN, ambas com valor igual a c:\app\oracle\instant
Falta ainda criar um usuário pra ti no banco de dados (o @Flavio Exterkoetter já está providenciando). Feito isto, iremos alterar o arquivo GestorIDSS.ini na tua máquina
Após isto, o Gestor IDSS já deve executar (usuário Samuel, senha Hoje.1608). A senha pode ser alterada depois em Arquivo, Alterar Senha.
